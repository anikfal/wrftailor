=========================
WRFtailor's Documentation
=========================

WRFtailor is a bunch of interactive tools for tailoring and modification of the
`Weather Research and Forecasting (WRF) <https://www.mmm.ucar.edu/weather-research-and-forecasting-model>`_
model input data, such as WRF domain files and emissions files. WRFtailor is based on the `NCAR Command Language (NCL) <https://www.ncl.ucar.edu>`_
and Linux shell scripts.

WRFtailor is available at:
 https://github.com/anikfal/wrftailor


.. note::

    WRFtailor is a Linux-based software, developed on Fedora/CentOS. However, it can be tested on the Mac OS.
